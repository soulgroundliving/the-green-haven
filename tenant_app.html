1. ข้อมูลที่เราเก็บรวบรวม

แอปพลิเคชัน Nature Haven เก็บข้อมูลต่อไปนี้เพื่อให้บริการแก่ผู้เช่า:

ข้อมูลส่วนตัว
• ชื่อ-นามสกุล, เบอร์โทรศัพท์, อีเมล, ทะเบียนรถยนต์
• หมายเลขห้อง และอาคารที่พักอาศัย
• วันเกิด (เพื่อคำนวณดวงชะตาประจำวัน — เก็บเฉพาะวันในสัปดาห์)
• ชื่อเล่นที่กำหนดเอง และรูปโปรไฟล์ (Avatar)

ข้อมูลสัตว์เลี้ยง
• ชื่อ, ประเภท, สายพันธุ์, เพศ, อายุ
• สถานะการฉีดวัคซีน, วันที่ฉีดและวันหมดอายุ

ข้อมูลการชำระเงิน
• ยอดบิลค่าเช่า, ค่าไฟ, ค่าน้ำ
• สลิปการโอนเงิน (ภาพ) สำหรับการยืนยันผ่าน SlipOK API
• Transaction Reference จาก SlipOK
• ข้อมูลบริษัท (ชื่อ, เลขประจำตัวผู้เสียภาษี, ที่อยู่) สำหรับใบเสร็จนามบริษัท

ข้อมูลการใช้งานและมิเตอร์
• ข้อมูลมิเตอร์ไฟฟ้า (หน่วยเก่า-ใหม่) และน้ำประปา
• ประวัติการใช้งานย้อนหลัง 6 เดือน

ข้อมูลการแจ้งและบริการ

1. พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล (PDPA) พ.ศ. 2562

เราดำเนินการตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 อย่างเคร่งครัด:
• เก็บรวบรวมข้อมูลเฉพาะที่จำเป็นต่อการให้บริการ (Data Minimization)
• ได้รับความยินยอมก่อนประมวลผลข้อมูลส่วนบุคคล
• แจ้งวัตถุประสงค์การใช้ข้อมูลอย่างชัดเจน
• มีมาตรการรักษาความปลอดภัยของข้อมูลที่เหมาะสม

2. ระบบจัดเก็บข้อมูล

• Google Firebase Firestore: เก็บข้อมูลผู้เช่า, บิล, สัตว์เลี้ยง, Marketplace, Wellness
• Google Firebase RTDB: ข้อมูล Real-time (มิเตอร์, การซ่อม, การทำความสะอาด, Gamification)
• Vercel (CDN): ให้บริการ Frontend Application
• เซิร์ฟเวอร์ทั้งหมดตั้งอยู่ในภูมิภาค Asia (Singapore/Tokyo)
• ข้อมูลถูกเข้ารหัสระหว่างการส่ง (TLS 1.3) และเมื่อจัดเก็บ (AES-256)

3. การเข้าถึงข้อมูล

• ผู้เช่า: เข้าถึงเฉพาะข้อมูลของตนเอง ผ่าน Firebase Security Rules
• แอดมิน: เข้าถึงข้อมูลทุกห้องผ่านการยืนยันตัวตน Firebase Auth
• ระบบอัตโนมัติ: Cloud Functions ประมวลผลตาม Business Logic ที่กำหนด
• ไม่มีพนักงานบุคคลภายนอกเข้าถึงข้อมูล

4. การยืนยันการชำระเงิน (SlipOK)

สลิปการโอนเงินถูกส่งผ่าน SlipOK API เพื่อยืนยันความถูกต้อง:
• ข้อมูลที่ส่ง: ภาพสลิปและจำนวนเงิน
• SlipOK ไม่เก็บข้อมูลส่วนตัวของผู้โอน
• Transaction Reference ถูกเก็บเพื่อการตรวจสอบย้อนหลัง

5. การแจ้งเหตุละเมิดข้อมูล
