นโยบายความเป็นส่วนตัว

มีผลบังคับใช้ตั้งแต่ 1 พฤษภาคม 2568 · The Green Haven Co., Ltd.

Nature Haven เก็บข้อมูลส่วนบุคคลเฉพาะที่จำเป็นต่อการให้บริการจัดการอาคารพักอาศัย โดยปฏิบัติตาม พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) อย่างเคร่งครัด

1. ข้อมูลที่เราเก็บรวบรวม

แอปพลิเคชัน Nature Haven เก็บข้อมูลต่อไปนี้เพื่อให้บริการแก่ผู้เช่า:

ข้อมูลส่วนตัว
• ชื่อ-นามสกุล, เบอร์โทรศัพท์, อีเมล, ทะเบียนรถยนต์
• หมายเลขห้อง และอาคารที่พักอาศัย
• วันเกิด (เพื่อคำนวณดวงชะตาประจำวัน — เก็บเฉพาะวันในสัปดาห์)
• ชื่อเล่นที่กำหนดเอง และรูปโปรไฟล์ (Avatar)

ข้อมูลยืนยันตัวตน (KYC — สำหรับการจองและทำสัญญาเช่า)
• รูปถ่ายบัตรประชาชน (ด้านหน้าและด้านหลัง)
• สำเนาทะเบียนบ้าน (ถ้ามี)
• หนังสือรับรองเงินเดือนหรือการทำงาน (ถ้ามี)
เก็บเฉพาะเพื่อยืนยันตัวตนผู้เช่าตอนจอง/ทำสัญญา และลบตามระยะเวลาในข้อ 5

ข้อมูลสัตว์เลี้ยง
• ชื่อ, ประเภท, สายพันธุ์, เพศ, อายุ
• สถานะการฉีดวัคซีน, วันที่ฉีดและวันหมดอายุ

ข้อมูลการชำระเงิน
• ยอดบิลค่าเช่า, ค่าไฟ, ค่าน้ำ
• สลิปการโอนเงิน (ภาพ) สำหรับการยืนยันผ่าน SlipOK API
• Transaction Reference จาก SlipOK
• ข้อมูลบริษัท (ชื่อ, เลขประจำตัวผู้เสียภาษี, ที่อยู่) สำหรับใบเสร็จนามบริษัท

ข้อมูลการใช้งานและมิเตอร์
• ข้อมูลมิเตอร์ไฟฟ้า (หน่วยเก่า-ใหม่) และน้ำประปา
• ประวัติการใช้งานย้อนหลัง 6 เดือน

ข้อมูลการแจ้งและบริการ
• คำขอแจ้งซ่อม (หมวดหมู่, รายละเอียด, รูปภาพ, ความเร่งด่วน)
• คำขอทำความสะอาด (วันที่, ช่วงเวลา, หมายเหตุ)
• การแจ้งนิติ/ร้องเรียน (ประเภท, ความรุนแรง, สถานที่, รายละเอียด)
• รูปภาพประกอบการแจ้งซ่อม

ข้อมูลชุมชนและ Marketplace
• โพสต์ Marketplace (ชื่อสินค้า, ราคา, รายละเอียด, รูปภาพ)
• LINE User ID (เมื่อผูกบัญชี LINE LIFF)

ข้อมูล Gamification
• คะแนน, ระดับ, ป้าย (Badge), ความสำเร็จ (Achievement)
• สถิติการ Login ต่อเนื่อง (Login Streak)
• ประวัติการอ่านบทความ Wellness และการรับคะแนน

2. วัตถุประสงค์การใช้ข้อมูล

• เพื่อให้บริการจัดการอาคารพักอาศัย เช่น ออกบิล, รับแจ้งซ่อม
• เพื่อยืนยันการชำระเงินและออกใบเสร็จ
• เพื่อการอนุมัติสัตว์เลี้ยงและบริการต่างๆ
• เพื่อส่งการแจ้งเตือนผ่าน LINE
• เพื่อระบบ Gamification สร้างแรงจูงใจในชุมชน

3. การแบ่งปันข้อมูล

• ผู้ดูแลระบบ (Admin): เข้าถึงข้อมูลทุกประเภทเพื่อการบริหารจัดการ
• SlipOK API: รับสลิปการโอนเงินเพื่อยืนยัน (ไม่เก็บข้อมูลบัตรเครดิต)
• LINE Corporation: ผ่าน LIFF SDK สำหรับการแจ้งเตือน
• Google Firebase: สำหรับการจัดเก็บและซิงค์ข้อมูลแบบ Real-time
• เราไม่ขายหรือเผยแพร่ข้อมูลส่วนตัวแก่บุคคลภายนอกเพื่อวัตถุประสงค์ทางการค้า

4. สิทธิ์ของเจ้าของข้อมูล (PDPA §30, §32)

ภายใต้ พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 คุณมีสิทธิ์:
• ขอสำเนาข้อมูล (§30): ขอให้ admin ส่งข้อมูลของคุณในรูปแบบ JSON
• แก้ไขข้อมูล: แก้ไขในแอปได้โดยตรง (ชื่อ, เบอร์, อีเมล, ทะเบียนรถ)
• ลบข้อมูล (§32): แจ้ง admin เพื่อดำเนินการ (ข้อมูลบัญชีที่กฎหมายบังคับเก็บจะถูก retain ไว้)
• คัดค้านการประมวลผล: แจ้ง admin

ติดต่อได้ผ่าน LINE Official หรือ admin@the-green-haven.com

5. ระยะเวลาการเก็บรักษาข้อมูล

• ใบตรวจสภาพห้อง (รูปภาพ, ลายเซ็น, บันทึก): ลบอัตโนมัติ 2 ปีหลังแอดมินเซ็นรับ
• ข้อมูลผู้เช่า (ชื่อ, ติดต่อ, gamification): ลบอัตโนมัติ 1 ปีหลังสิ้นสุดสัญญา
• บิล, รายการชำระเงิน: เก็บ 5 ปี ตามประมวลรัษฎากร มาตรา 87
• สัญญาเช่า: เก็บ 5 ปี ตาม ป.พ.พ. มาตรา 193/34

ข้อมูลที่ไม่มีวัตถุประสงค์ที่ชัดเจนจะถูกลบอัตโนมัติภายใน 5 ปีจากวันที่สร้าง
