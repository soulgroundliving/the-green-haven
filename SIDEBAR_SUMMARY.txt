CUSTOMIZATION
================================================================================

Change Sidebar Width:
  Modify: .sidebar { width: XXXpx; }
          .main-with-sidebar { margin-left: XXXpx; }

Change Colors:
  Modify: --green, --green-dark, --green-pale in :root

Change Animation Speed:
  Modify: transition: all 0.3s ease; (0.3s = fast, 0.5s = slow)

Change Fonts:
  Modify: font-family: 'Your Font', sans-serif;

Add More Nav Items:
  Copy: .nav-item button structure
